Two available sessions

You must choose, but choose wisely.

For this month there are two different workshoppers available. Each offers a very different set of lessons but each will provide the same valuable experience. Please try to have Node.js and the workshopper that you choose installed BEFORE the night of the session.

  1. Level Up your JS: Want to learn valuable skills and concepts that can be used in Node.js or in the browser then choose Functional Javascript. This will provide you with a fundamental functional programming knowledge and you will learn about functional ES5 features like map and reduce.
  2. New to node: If you already know JavaScript and are ready to dive head first into Node.js it is best to start with the workshopper Learnyounode. Most web developers will find that this a great intro into the Node.js platform.

Protip: As you work through these sessions create a folder for each workshopper. Each workshopper lesson will require its own file for testing. When you approach a new lesson create a new file. At the end of the workshop you should then have a large list of programs that you will be able to reference in the future.

Getting Started with Node

Node.js is easy to install. Just visit the download page and get started. (https://nodejs.org/en/download/)

The official site has great documentation on their APIs. (https://nodejs.org/docs/v5.1.0/api/)

Mozilla Developer Network probably has the best JavaScript docs. (https://developer.mozilla.org/en-US/docs/Web/JavaScript)

The Art of Node is a great guide on the core concepts of Node.js. (https://github.com/maxogden/art-of-node/#the-art-of-node)

An excellent list of modules, resources, and other node knowledge is listed on Awesome Node. (https://github.com/sindresorhus/awesome-nodejs)

Learnyounode

Github: https://github.com/workshopper/learnyounode

Ensure that Node.js is installed and that you have an internet connection. Open your terminal and run this command: 

npm install --global learnyounode

The --global option installs this module globally so that you can run it as a command in your terminal.

Having issues with installation?

If you get an EACCESS error, the simplest way to fix this is to rerun the command, prefixed with sudo: 

sudo npm install --global learnyounode

You can also fix the permissions so that you don't have to use sudo. Take a look at this npm documentation: https://docs.npmjs.com/getting-started/fixing-npm-permissions

Run the workshop

Open your terminal and run the following command: 

learnyounode

Tips on approaching the problems

All solutions should be around 30 lines or less, most are less than 15 lines of code. The complexity is around understanding which javascript functions or function symantics should be used. No modules needed, just pure JS.

Recursion - You will need to create an additional helper function although this is not mentioned in the documentation.

Block Event Loop - Be sure to reference the MDN documentation linked to.
